# 🗄️ Estudos em SQL - Banco de Dados

Repositório dedicado aos meus estudos em **SQL**, com foco em manipulação e gerenciamento de bancos de dados relacionais.

## 🚀 Objetivo

Aprender e aplicar conceitos fundamentais de banco de dados e consultas SQL.

## 🛠️ Tecnologias utilizadas

- SQL
- MYSQL Server
- SQLITE

## 📚 Conteúdos estudados

### 🔹 Modelagem e Estrutura
- Criação de tabelas (`CREATE TABLE`)
- Chaves primárias (Primary Key)
- Chaves estrangeiras (Foreign Key)
- Constraints

### 🔹 Manipulação de Dados
- `SELECT`
- `INSERT`
- `UPDATE`
- `DELETE`
- `WHERE`

### 🔹 Consultas Avançadas
- `INNER JOIN`
- `RIGHT JOIN`
- `UNION`
- `GROUP BY`

### 🔹 Programação no Banco
- Procedures
- Functions
- Triggers

### 🔹 Controle de Transações
- `TRANSACTION`
- `ROLLBACK`

## 📌 Aprendizados

- Modelagem de banco de dados
- Escrita de consultas eficientes
- Manipulação e controle de dados
